# Data

This directory contains all experimental data.

## Files

- test_inputs.py  
  Input prompts used in experiments

- rem_experiment_results.csv  
  Raw model outputs (control, pre, post)

- rem_analysis_results.csv  
  Processed metrics (lift, similarity, etc.)

## Regeneration

All data can be regenerated by running:

python3 code/pipeline/rem_experiment.py  
python3 code/analysis/analyze_results.py
